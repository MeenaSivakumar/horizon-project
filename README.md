# 🌟 Introduction to Styled-Components in React

`styled-components` is a popular library for styling React applications.  
It lets you write **CSS in JavaScript**, scoped to components, making styles **modular, reusable, and dynamic**.

---

## 🚀 Why Styled-Components?

- **Scoped Styles** → No CSS conflicts across files  
- **Dynamic Styling with Props** → Pass props to change styles  
- **Conditional & Themed Styles** → Easily integrate with design systems  
- **Global Styles Support** → Still manage resets and base styles  

---

## 📌 Features Learned

- Styling with **props**  
- Adding **hover & active states**  
- Creating **global styles**  
- Building **reusable components** like Button, Input, and Card  

---

## 🧑‍💻 Components Built

- ✅ **Button** → Reusable button with dynamic props (`primary`, hover, active)  
- ✅ **Input** → Styled input with focus effects  
- ✅ **Card** → Container with title, subtitle, hover animation
